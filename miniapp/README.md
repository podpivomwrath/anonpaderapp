# Мини-апп «Монолит» — хаб персонажа

VK Mini App: React + Vite + [@vkontakte/vkui](https://vkui.io) + [@vkontakte/vk-bridge](https://github.com/VKCOM/vk-bridge).
Делит базу данных с ботом (общий Postgres, тот же процесс aiohttp-сервера
из [main.py](../main.py) — эндпоинты `/api/miniapp/*` в [bot/miniapp_api.py](../bot/miniapp_api.py)).

Готова вкладка «Характеристики» (распределение статов, живой предпросмотр
производных характеристик). Инвентарь/Пресеты/Биржа — заглушки «Скоро».

## Разработка

```bash
cd miniapp
npm install
npm run dev
```

Дев-сервер (порт 5173) проксирует `/api/miniapp/*` на `http://localhost:8080`
(см. `vite.config.js`) — бот (`python main.py` в корне репозитория) должен
быть запущен рядом.

Открывать мини-апп напрямую по `http://localhost:5173` без VK окружения
можно, но бэкенд ответит 403 — VK передаёт подписанные launch-параметры
(`vk_user_id`, `sign`, ...) в query-строке только сам, при открытии из
приложения ВКонтакте. Для локальной проверки без реального VK можно
сформировать подпись вручную тем же секретом, что в `.env` (`VK_MINIAPP_SECRET`),
алгоритмом из `bot/miniapp_auth.py`, и открыть
`http://localhost:5173/?vk_user_id=<id>&vk_app_id=1&sign=<подпись>`.

## Настройка приложения в VK

1. Зайти в [vk.com/editapp](https://vk.com/editapp?act=create), создать
   **VK Mini App** (не сообщество и не standalone-приложение).
2. В настройках приложения указать **базовый домен/URL** — адрес, откуда
   отдаётся собранный фронтенд (см. «Продакшен» ниже). Для разработки можно
   указать туннель (ngrok и т.п.) до `localhost:5173`.
3. Мини-апп работает только по **HTTPS** — как и Callback API бота.
4. Скопировать **ID приложения** (APP_ID) и **секретный ключ** приложения
   (Настройки → «Защищённый ключ», НЕ путать с Callback-секретом бота).
5. Прописать в `.env` бота (корень репозитория, не `miniapp/.env`):
   ```
   VK_MINIAPP_SECRET=<секретный ключ приложения>
   VK_MINIAPP_URL=https://vk.com/app<APP_ID>
   VK_MINIAPP_ORIGIN=<домен, откуда отдаётся фронтенд>
   ```
   `VK_MINIAPP_URL` — то, на что ведёт кнопка «🎭 Персонаж» в боте
   (появляется автоматически, как только `VK_MINIAPP_URL` не пустой).
   `VK_MINIAPP_ORIGIN` — CORS-origin фронтенда (можно `*` на время разработки).

## Продакшен

```bash
cd miniapp
# если бэкенд на другом домене, чем статика фронтенда:
echo "VITE_API_BASE_URL=https://your-bot-domain.example.com" > .env
npm run build
```

Собранная статика (`miniapp/dist/`) заливается на хостинг, указанный как
базовый домен приложения в vk.com/editapp (VK поддерживает свой статический
хостинг для Mini Apps, либо можно указать свой сервер).
